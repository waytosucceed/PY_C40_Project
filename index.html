Full Body Detection Using Haar Cascade

In this project, your task is to detect full human bodies inside a given image using a Haar Cascade classifier. You‚Äôll scan the image, find every full body, draw a red rectangle around it, and display the result.
This is exactly how basic pedestrian detection systems work, in smart surveillance, crowd analytics, and even retail footfall tracking.

  • Use a new Haar Cascade file trained to detect full human figures
  • Load and convert the image to grayscale
  • Run the detector
  • Draw red rectangles around each detected body
  • Display the final output

Here are the steps you need to follow:

Step 1: Setup the Project

  • Download the project template
  • Open the folder in Visual Studio Code.
  • Inside it, create a new Python file named project.py

Step 2: Download the Haar Cascade for Full Body Detection

  • Visit the official Haar Cascade repository:
    https://github.com/opencv/opencv/tree/master/data/haarcascades
  • Scroll and find the file:
    haarcascade_fullbody.xml
  • Click on the file name > Click the "Raw" button (top right of the code box) > Right-click anywhere and select ‚ÄúSave As...‚Äù > Save it in your project folder.

Step 3: Import the Required Library
Import the cv2 library in your Python file to access OpenCV's tools.

Step 4: Load the Image
Load the img.jpg image using cv2.imread() and save it in a variable called image.

Step 5: Load the Haar Cascade Classifier

  • Use cv2.CascadeClassifier('haarcascade_fullbody.xml') to load the model.
  • Store it in a variable named classifier.

Step 6: Convert Image to Grayscale

  • Convert the original image to grayscale using cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
  • Store it in a variable named grayscale_image.

Step 7: Detect Full Bodies in the Image

  • Use the detectMultiScale() function on the grayscale image to detect bodies: classifier.detectMultiScale(grayscale_image, scaleFactor=1.04, minNeighbors=5)
  • Save the result in a variable named bodies.

Step 8: Draw Rectangles Around the Detected Bodies

  • Use a for loop to draw red rectangles around each detected body:
    cv2.rectangle(image, (x, y), (x + w, y + h), (0, 0, 255), 2)
  • Loop over the variable bodies and extract (x, y, w, h) from each item.

Step 9: Display the Final Output

  • Use cv2.imshow() to show the image with rectangles.
  • Use cv2.waitKey(0) to keep the window open until a key is pressed.
